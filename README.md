# ReaderPro.io

## What is ReaderPro.io?
ReaderPro.io is a free, browser-based door access programming tool built for security systems professionals. It gives you a clean visual interface to plan and document your entire access control layout — controllers, sub-panels, readers, locks, and door hardware — all in one place, before you ever touch a panel.

No installs, no accounts, no vendor lock-in. Designed to be vendor agnostic it works with Lenel, Software House, Genetec, Alta and any other platform you program against. When you're done, export your schedule as a PDF, Excel spreadsheet, YAML, or JSON — or share it instantly with a single link.

## Who is ReaderPro.io for?
ReaderPro.io is built for security systems integrators, application engineers, and field technicians — anyone responsible for programming and documenting access control systems. If you've ever pre-planned a door schedule in a spreadsheet, scribbled reader addresses on a napkin, or dug through a job folder trying to remember what got commissioned and what didn't, this tool was made for you.

## How do I use ReaderPro.io?
Using ReaderPro.io is as simple as opening a browser. There's nothing to install, no account to create, and no internet connection required after the page loads. Just open the file locally and get to work — which makes it especially useful when you're tucked in an IDF closet, a mechanical room, or anywhere else on a job site where Wi-Fi is an afterthought.

Start by entering your project details at the top — job number, site, platform, and technician info. From there, add your controllers one at a time. For each controller, fill in the network configuration (IP, subnet, gateway), then build out your door schedule row by row: door name, zone, lock type, reader type, reader address, REX, snd DPS. If your controller uses sub-panels or reader boards, you can add those too and nest the doors underneath — keeping your addressing clean and visual.

As you work, the output panel at the bottom updates live in YAML and JSON so you always have a structured record of your configuration. When you're ready to hand something off or archive the job, export it as a PDF door schedule, an Excel spreadsheet, or a shareable link that encodes the entire project in the URL — no server, no file to email.
